# samplepacks/


a samplepack contains all the audio to be used together on a project.  
a samplepack is simply a folder that contains subfolders of wav files.

```
bop/
	samplepacks/
		\[samplepack001]/
			\[folder001]/
				wave001.wav
				wave002.wav
				wave003.wav
			\[folder002]/
				wave001.wav
				wave002.wav
				...
		\[samplepack002]/
			\[folder001]/
				wave001.wav
				wave002.wav
				wave003.wav
			\[folder002]/
				wave001.wav
				wave002.wav
				...			
```
