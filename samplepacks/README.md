# samplepacks/

_do not delete .gitignore or README.md_


A samplepack is a folder that contains folders of wav files - it might contain all the audio to be used together on a project or a piece. 



```
bop/
	samplepacks/
		.gitignore
		README.md

		example_samplepack/
			folder1/
				wave001.wav
				wave002.wav
				wave003.wav
			folder2/
				wave001.wav
				wave002.wav
				...

		my_cool_samplepack/
			000drumkit/
				001.wav
				002.wav
				003.wav
			001pno_multisample/
				001c3.wav
				002c4.wav
				...			
```
